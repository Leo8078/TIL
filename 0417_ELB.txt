ELB : AWS에서 제공하는 트래픽 분산 서비스
만약 사용자가 관리 할 수 없을 정도로 EC2가 많다면? -> ELB 사용

Elastic Load Balancer

내결합성 
느슨한 결합 / 다른 하나의 서비스가 부하가 생겨도 전체에는 영향을 크게 미치지 않음

ELB 특징
- 다수의 서비스에 트래픽을 분산

- Health Check : 직접 트래픽을 발생시켜 Instance가 살아있는지 체크

- Autoscaling 과 연동 가능

- 지속적으로 IP 주소 변동 / 고정 불가능 -> 도메인(주소) 기반 사용

총 3가지 종류
A(pplication) L B - 똑똑함
N(etwork) L B - 빠름
C(lassic) L B - 옛날(구식)

ALB 
타겟 그룹 기준으로 요청 분배
EC2 인스턴스, IP 주소, Lambda 함수
